HTML stands for HyperText Markup Language. It is used to create and structure webpages.

Structure of an HTML Page

Every HTML page starts with a <!DOCTYPE html> declaration, followed by <html>, <head>, and <body> tags.

Example:

<!DOCTYPE html>
<html>
   <head>...</head>
   <body>...</body>
</html>
